# flex-box model을 배워보자
## container(부모 속성)
```css
display: flex;
justify-content: flex-start;           /* 왼쪽으로 정렬[기본값] */
justify-content: flex-end;             /* 오른쪽으로 정렬 */
justify-content: center;               /* 가운데 정렬 */
justiry-content: space-between;        /* 양 끝 정렬 후 균등분할 */
justiry-content: space-around;         /* 자식 주변으로 균등분할 */
justiry-content: space-evenly;         /* 균등분할 */

align-items: stretch;     /* [기본값] 자식중 높이가 가장 큰 자식에게 모든 자식을 맞춘다.*/
align-items: center;     /* 가운데 맞춘다 */
align-items: flex-start;     /* 위에 맞춘다 */
align-items: flex-end;     /* 아래에 맞춘다 */
```
