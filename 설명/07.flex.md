# flex-box model을 배워보자

[css-tricks flex설명](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)

## container(부모 속성)
```css
display: flex;
justify-content: flex-start;           /* 왼쪽으로 정렬[기본값] */
justify-content: flex-end;             /* 오른쪽으로 정렬 */
justify-content: center;               /* 가운데 정렬 */
justiry-content: space-between;        /* 양 끝 정렬 후 균등분할 */
justiry-content: space-around;         /* 자식 주변으로 균등분할 */
justiry-content: space-evenly;         /* 균등분할 */

align-items: stretch;     /* [기본값] 자식중 높이가 가장 큰 자식에게 모든 자식을 맞춘다.*/
align-items: center;       /* 가운데 맞춘다 */
align-items: flex-start;   /* 위에 맞춘다 */
align-items: flex-end;     /* 아래에 맞춘다 */

flex-direction: row | row-reverse | column | column-reverse
/* column 일 경우에는 justify-content, align-items가 바뀐다. */

flex-wrap: nowrap | wrap | wrap-reverse
/* flex는 기본이 넘치지 않는다(자식의 width를 무시하면서 한줄에 넣는다.) */
/* 자식요소가 넘치게(여러줄) 하려면 wrap 속성을 줘야 한다. */

align-content: stretch | flex-start | flex-end | space-between | space-around | space-evenly
/* 부모의 세로크기가 자식들보다 클 때, 그리고 자식들이 여러줄로 표현될 때 각 줄들을 어떻게 배치하는지에 대한 속성 */
```
